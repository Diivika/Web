# 💇‍♂️ BeautyWeb - Онлайн-запись на стрижку

> **BeautyWeb** - это веб-приложение, которое позволяет клиентам легко и быстро записаться на желаемую стрижку к
> барберу. Проект реализован на Python с использованием веб-фреймворка Flask, базы данных SQLite и современных
> фронтенд-технологий.

## 📖 Описание проекта

**BeautyWeb** создан для автоматизации записи клиентов в парихмарискую. Сайт решает следующие задачи:

- Предоставляет клиентам удобный каталог барберов и услуг.
- Позволяет записаться онлайн без звонков и очередей.
- Даёт барберам возможность регистрироваться в системе через администратора.

Проект ориентирован на небольшие и средние парикмахерские, которые хотят иметь современный инструмент для работы с
клиентами.

---

## ✨ Основные возможности

### Для клиентов:

- 🔐 **Регистрация и авторизация** (email/пароль).
- ✂️ **Просмотр списка барберов** с фотографиями .
- 📍 **Просмотр места стрижки** на карте и адреса.
- 🔔 **получение напоминаний** (telegram).

### Для администратора:

- 🧑‍💼 **Модерация регистрации барберов**.

## 🚀 Установка и запуск

### Требования

Перед установкой убедитесь, что на вашем компьютере есть:

- **Python** версии 3.9 или выше.
- **pip** (менеджер пакетов Python).
- **Git** (опционально, для клонирования репозитория).
- **Виртуальное окружение** (рекомендуется).

### Инструкция по установке

1. **Клонируйте репозиторий** (или скачайте ZIP-архив):
   ```bash
   git clone http://https://github.com/Diivika/Web
   cd BeautyWeb
2. **установите зависимости через cmd**
    ```bash
   pip install -r requirements.txt
3. **создайте папку db в корневой папке сайта*
    ```bash
   mkdir db
4. **запустите файл add_all.py**
    ```bash
   python add_all.py
5. **запустите main.py с желаемыми параметрами**
    ```bash
    python main.py --port 8080 --debug
